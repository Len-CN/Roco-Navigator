# 洛克王国导航助手

> **声明**：本项目使用 AI 辅助开发（Claude Code），所有代码经人工审查。

一个洛克王国的地图导航工具。它通过截取屏幕上的小地图，自动识别你在游戏世界中的位置，帮你规划路线、找资源点。

**安全说明**：只是"看"你的屏幕，不会操作你的鼠标键盘，也不会动游戏本身。

## 能做什么

- **自动定位** — 开着游戏走路，工具实时显示你在大地图上的位置
- **资源点标注** — 地图上显示各种采集点、宝箱、精灵分布等（数据来自 WIKI）
- **路线规划** — 选一堆想去的点，自动算出最短路线
- **悬浮窗导航** — 游戏时旁边飘一个小地图，箭头指引你往哪走
- **点位筛选** — 只看你想看的类型，比如只显示宝箱或只显示采集点

## 怎么用

### 第一步：启动

双击 `start.bat` 就行。第一次会自动装环境和依赖（需要几分钟），之后秒开。

> 前提：电脑上装了 [Python 3.10+](https://www.python.org/downloads/)，安装时记得勾 **Add to PATH**。

### 第二步：下载数据

打开程序后，在左边侧边栏：
1. 点 **更新地图** — 下载游戏大地图（约 480 张瓦片，需要几分钟）
2. 点 **更新点位** — 下载资源点坐标和图标

这一步只需做一次，数据会保存在本地。

### 第三步：校准小地图

点 **校准小地图**，用鼠标框选游戏窗口右上角的那个圆形小地图区域。这样程序才知道去哪里截图。

### 第四步：开始追踪

点 **开始**，程序就会实时追踪你的位置了。地图上会出现一个红色箭头表示你在哪、朝哪个方向。

### 第五步：规划路线（可选）

在地图上勾选你想去的资源点，点 **开始规划**，程序会算出最优路线。再点 **开始导航**，悬浮窗会显示方向指引。

## 手动安装（不用 start.bat）

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

## 可选增强

装了这些会更强，不装也能正常用：

| 装什么 | 有什么用 | 怎么装 |
|--------|----------|--------|
| `torch` + `kornia` | AI 识别模式，抗干扰更强 | `pip install torch kornia` |
| `ortools` | 大量点位时路线更优 | `pip install ortools` |
