# 洛克王国导航助手

> **声明**：本项目使用 AI 辅助开发（Claude Code）。不保证功能可用性和稳定性，使用风险自负。本工具仅供学习交流，因使用本工具导致的任何后果（包括但不限于游戏账号封禁、数据丢失等）均由使用者自行承担，开发者不承担任何责任。

洛克王国纯视觉导航工具。通过屏幕截图读取游戏小地图，实时识别玩家在世界地图中的位置，支持资源点路线规划和 HUD 悬浮窗导航。

仅读取屏幕像素，不注入游戏进程，不模拟键盘鼠标。

## 功能

- **实时定位** — 基于 SIFT 特征匹配，预计算地图特征 + 空间索引，实时追踪玩家位置和朝向
- **多策略降级** — SIFT → LoFTR (GPU) → 颜色检测 → 惯性预测，确保持续可用
- **路线规划** — 选择目标点位后自动计算最优路线（方向扫描/OR-Tools/最近邻）
- **HUD 悬浮窗** — 半透明浮动小地图，显示资源图标和方向指引，支持拖拽/缩放/穿透点击
- **WIKI 集成** — 自动从 Bilibili WIKI 下载大地图瓦片、资源点位坐标和图标
- **点位筛选** — 按标记类型分类显示或隐藏资源点

## 快速开始

### 环境要求

- Windows 10/11
- [Python 3.10+](https://www.python.org/downloads/)（安装时勾选 Add to PATH）

### 启动

双击 `start.bat`，首次运行自动创建虚拟环境并安装依赖。

手动安装：

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### 使用流程

1. **更新地图** — 侧边栏点击"更新地图"，下载并拼接世界地图瓦片
2. **更新点位** — 点击"更新点位"，获取资源坐标和图标数据
3. **校准小地图** — 点击"校准小地图"，框选游戏窗口中的小地图区域
4. **开始追踪** — 点击"开始"，程序实时定位并在地图上显示玩家位置
5. **路线规划** — 勾选目标点位，点击"开始规划"，再点"开始导航"启用悬浮窗指引

### 可选依赖

基础功能无需额外安装。以下依赖可增强特定功能：

| 依赖 | 用途 | 安装 |
|------|------|------|
| `torch` + `kornia` | LoFTR 深度学习匹配，抗干扰更强（需 GPU） | `pip install torch kornia` |
| `ortools` | Google OR-Tools 求解器，大量点位时路线更优 | `pip install ortools` |

未安装时自动降级到基础算法，不影响核心功能。

## 技术简介

```
屏幕截图 → 预处理(CLAHE+圆环遮罩) → 多策略定位 → EMA平滑 → 路线规划 → HUD导航
```

- **定位**：启动时对整张地图提取 SIFT 特征并建立 256px 网格空间索引；运行时仅对小地图提取特征，在网格邻域内快速匹配
- **追踪**：全局扫描 → 精确追踪 → 惯性导航，三阶段自适应切换
- **路线**：方向扫描算法（PCA 主轴 → 自适应条带分割 → 蛇形遍历 → 2-opt 优化），有 OR-Tools 时自动使用最优求解
- **坐标系**：基于 Bilibili WIKI Leaflet CRS.Simple，zoom 7 地图 (6144×5120)
